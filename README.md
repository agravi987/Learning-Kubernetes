# Kubernetes Learning Journey 🚀

Welcome to your Kubernetes practice repo.

This project is organized to take you from zero to confident, step by step, without making the learning feel boring.

## 📦 What is inside

| Folder | What you learn |
| :--- | :--- |
| `0. Kubernetes From Zero` | A friendly starting guide and task checklist |
| `1. Kubernetes-basics` | Core concepts like cluster, node, pod, deployment, and service |
| `2. Important Basic command Used` | The essential `kubectl` commands |
| `3. Deployment and Running of nginx` | Your first real app deployment |
| `4. Rollout and Rollback` | Safe updates, version history, and rollback |
| `5. Self Healing and Scaling and Yamls` | Self-healing, scaling, and YAML practice |

## 🎯 Recommended Learning Order

1. Start with `0. Kubernetes From Zero`
2. Read `1. Kubernetes-basics`
3. Practice `2. Important Basic command Used`
4. Try `3. Deployment and Running of nginx`
5. Learn `4. Rollout and Rollback`
6. Finish with `5. Self Healing and Scaling and Yamls`

## 🧠 How to study

Use this loop:

1. Read one topic
2. Run the commands yourself
3. Break something safely
4. Fix it
5. Repeat until it feels natural

## ⭐ Goal

By the end, you should be able to:

- explain Kubernetes in simple words
- use `kubectl` without fear
- deploy and expose apps
- update apps safely
- scale when needed
- understand self-healing
